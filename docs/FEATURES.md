# 功能设计

## 一、系统概述

BioPathoFocus 是面向科研用户的微生物基因组数据库展示系统，提供菌株数据的可视化浏览与分析能力。系统采用工作台式布局，顶部导航栏提供模块快速切换，左侧边栏提供筛选与快捷操作，右侧为主内容区。

## 二、布局设计

### 2.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  TheHeader: ☰ Logo │ 导航按钮(6个) │ 面包屑 │ 用户信息  │
├──────────┬──────────────────────────────────────────────┤
│          │                                              │
│ TheSidebar│           Main Content                      │
│  功能导航  │         (router-view)                      │
│  快捷操作  │                                              │
│          │                                              │
│  版本信息  │                                              │
├──────────┴──────────────────────────────────────────────┤
```

### 2.2 顶部导航栏 (TheHeader)

- **左侧**：汉堡菜单按钮（控制侧边栏折叠）+ Logo（"B" 图标 + "BioMe DB" 文字）
- **中间**：6 个模块导航按钮，当前激活模块高亮显示（蓝色背景 + 阴影）
- **右侧**：面包屑导航（根据 `route.matched` 自动生成）+ 分隔线 + 用户名 + 头像

### 2.3 左侧边栏 (TheSidebar)

- **展开状态**：宽度 `w-52`，显示图标 + 文字
- **折叠状态**：宽度 `w-14`，仅显示 emoji 图标
- **功能导航区**：与顶部导航相同的 6 个模块入口
- **快捷操作区**：数据筛选、导出数据、分析历史（预留）
- **底部**：版本号 + Mock 模式标识

## 三、功能模块

### 3.1 数据集概览 (Dashboard)

**路径**：`/dashboard`

**功能描述**：系统首页，展示数据集全局统计概览。

**页面结构**：

| 区域 | 布局 | 内容 |
|------|------|------|
| KPI 统计卡片 | 4 列 grid | 总样本数、覆盖国家、耐药基因数、毒力基因数 |
| 第一行图表 | 2 列 grid | 全球菌株地理分布（散点图）、菌株时间分布（柱状图） |
| 第二行图表 | 3 列 grid | 国家分布（饼图）、耐药基因统计（饼图）、毒力基因统计（饼图） |
| 第三行统计 | 3 列 grid | 宿主/样本类型/分离源分布（进度条列表） |

**数据源**：并行请求 6 个 API 接口（`Promise.all`）

**图表详情**：
- **地理分布散点图**：ECharts scatter 类型，xAxis/yAxis 隐藏，气泡大小 = `Math.sqrt(count) * 4`
- **时间线柱状图**：年份 vs 菌株数，蓝色圆角柱体
- **饼图**：环形图 `radius: ['40%', '68%']`，8 色调色板，图例右侧垂直排列
- **属性进度条**：宽度按 `Math.min(100, (count / maxCount) * 100)` 计算

---

### 3.2 样本信息 (Sample)

**路径**：`/sample`

**功能描述**：样本数据的浏览与多维筛选，支持 4 个 Tab 切换。

**Tab 结构**：

| Tab | 路径 | 功能 |
|-----|------|------|
| CSV | `/sample/csv` | 样本列表，支持多维筛选 + 分页 |
| MLST | `/sample/mlst` | MLST 分型数据表格 |
| cgMLST | `/sample/cgmlst` | cgMLST 等位基因矩阵 |
| 血清抗原 | `/sample/antigen` | 血清抗原分型数据表格 |

#### 3.2.1 CSV 样本列表

- **筛选功能**：样本类型（下拉）、宿主（下拉）、采样国家（输入）、分离菌（输入）
- **激活筛选标签**：已选筛选条件以 Tag 形式展示，支持单独移除
- **表格列**：样本编号、样本名称、样本类型、分离菌、ST 型、血清型、宿主、采样国家、分离时间、分离机构（共 10 列）
- **排序**：样本编号、ST 型、分离时间支持排序
- **分页**：每页 20 条，支持翻页

#### 3.2.2 MLST 分型

- 简单表格视图，列：样本编号、样本名称、Scheme、ST 型
- 支持分页

#### 3.2.3 cgMLST 等位基因矩阵

- 使用 GeneMatrix 组件渲染基因 x 样本的二维矩阵
- 行 = 基因名（locus），列 = 样本 ID，值 = 等位基因编号
- 有值单元格蓝色高亮，无值灰色显示

#### 3.2.4 血清抗原

- 简单表格视图，列：样本编号、K-type、O-type、H-type
- 支持分页

---

### 3.3 分型信息 (Typing)

**路径**：`/typing`

**功能描述**：独立的分型分析页面，提供更详细的分型数据展示。

**Tab 结构**：

| Tab | 路径 | 功能 |
|-----|------|------|
| MLST | `/typing/mlst` | 表格/饼图双视图切换 + ST 型筛选 |
| cgMLST | `/typing/cgmlst` | cgMLST 等位基因矩阵 |
| 表面抗原 | `/typing/antigen` | K/O/H-type 三维筛选 + 表格 |

#### 3.3.1 MLST 分型

- **双视图模式**：通过 BaseTabs 切换"详情表"和"饼图"
- **详情表**：标准 DataTable + 分页，支持 ST 型筛选
- **饼图**：展示 ST 型分布占比

#### 3.3.2 cgMLST 等位基因矩阵

- 与样本页 cgMLST Tab 相同的矩阵视图

#### 3.3.3 表面抗原

- **三维筛选**：K-type、O-type、H-type 三个输入框
- DataTable + 分页

---

### 3.4 耐药基因 (AMR)

**路径**：`/amr`

**功能描述**：耐药基因分析，支持矩阵浏览和统计图表两种视图。

**视图模式**：

| 模式 | 内容 |
|------|------|
| 详情表 | GeneMatrix 基因-样本矩阵（0/1 存在/缺失），支持类别和基因筛选 + 分页 |
| 统计图 | AMR 类别分布柱状图 + Top 耐药基因携带率柱状图 |

**筛选字段**：
- AMR 类别（下拉选择，动态加载选项）
- AMR 基因（文本输入）

**数据转换**：API 返回 `{ geneName, samples: { [sampleId]: 0|1 } }` → GeneMatrix 所需的 `genes/samples/matrix` 格式

---

### 3.5 毒力基因 (VF)

**路径**：`/vf`

**功能描述**：毒力基因分析，结构与 AMR 模块对称。

**视图模式**：

| 模式 | 内容 |
|------|------|
| 详情表 | GeneMatrix 基因-样本矩阵，支持类别和基因筛选 + 分页 |
| 统计图 | VF 类别分布柱状图 + Top 毒力基因覆盖率柱状图 |

**筛选字段**：
- VF 类别（下拉选择，动态加载选项）
- VF 基因（文本输入）

---

### 3.6 Tree Heatmap

**路径**：`/tree-heatmap`

**功能描述**：MLST 最小生成树与基因分布热力图联合分析。

**页面结构**：

| 区域 | 布局 | 内容 |
|------|------|------|
| 分析配置 | 1/4 宽度 | 标记信息选择、MLST Scheme、样本/基因范围、应用配置按钮 |
| 热力图 | 3/4 宽度 | HeatmapChart 基因分布热力图 |

**导出功能**：支持 SVG、PNG、数据三种导出方式

**热力图配置**：
- 颜色范围：`['#f0f5ff', '#1890ff']`（浅蓝到深蓝）
- X 轴标签旋转 45 度
- visualMap 水平放置在底部

## 四、公共组件

### 4.1 基础 UI 组件 (common/)

| 组件 | 功能 | 关键特性 |
|------|------|---------|
| BaseButton | 按钮 | primary/ghost 变体，sm/md 尺寸 |
| BaseCard | 卡片容器 | 标题 + header-extra 插槽，可选 padding |
| BaseInput | 文本输入 | v-model 双向绑定，focus 蓝色边框 |
| BaseSelect | 下拉选择 | 自定义下拉面板，鼠标离开关闭 |
| BaseTabs | Tab 切换 | 下划线式，蓝色激活指示器 |
| BaseTag | 标签 | 药丸样式，可选 closable |
| BasePagination | 分页 | 含省略号算法（总页数 > 7 时智能折叠） |
| BaseLoading | 加载状态 | 旋转 spinner + 文字提示 |
| BaseEmpty | 空状态 | 空箱子图标 + 描述文字 |
| BaseModal | 模态框 | Teleport 到 body，遮罩层点击关闭 |
| BaseTooltip | 文字提示 | CSS-only，hover 显示 |

### 4.2 图表组件 (chart/)

| 组件 | 图表类型 | Props |
|------|---------|-------|
| ChartContainer | 通用容器 | `option`, `autoResize` |
| BarChart | 柱状图 | `data`, `title`, `horizontal` |
| PieChart | 环形饼图 | `data`, `title` |
| HeatmapChart | 热力图 | `xLabels`, `yLabels`, `data`, `title` |

### 4.3 表格组件 (table/)

| 组件 | 功能 | 关键特性 |
|------|------|---------|
| DataTable | 通用数据表 | 列定义、固定表头、排序标识、动态插槽、最大高度滚动 |
| GeneMatrix | 基因矩阵 | 首列固定(sticky)、有值蓝色高亮、虚拟滚动 |

### 4.4 筛选栏 (filter/FilterBar)

- 卡片式背景容器
- 默认 slot 放置筛选控件
- 内置搜索（蓝色）和重置（灰色边框）按钮
- actions slot 放置额外操作按钮

## 五、交互设计

### 5.1 导航

- 顶部导航栏和侧边栏均可切换模块
- 当前激活模块在两处同步高亮
- 面包屑自动根据路由层级生成

### 5.2 筛选

- 筛选条件以 Tag 形式展示在筛选栏上方
- 每个 Tag 支持单独关闭（移除对应筛选条件）
- 重置按钮一键清空所有筛选条件

### 5.3 表格

- 固定表头，内容区滚动
- 排序列显示箭头指示器
- 支持单元格自定义渲染（具名插槽）
- 分页器显示总条数和页码

### 5.4 图表

- 所有图表自动适应容器尺寸（ResizeObserver）
- 饼图 hover 显示标签
- 热力图 emphasis 添加阴影效果
- Tree Heatmap 支持 SVG/PNG/数据导出
